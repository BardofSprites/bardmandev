What is a Graph

Graphs consist of nodes and edges. They are useful for showing relationships between things/people.

Think of graph as social network on FaceBook (spooky ðŸ‘»). You have certain attributes/data about you that is contained in your node, and you have groups of friends (also nodes with attributes).

The relationships shown in the edges can be directional (for example direction of hatred, this person hates this person) or can be undirectional (both nodes are colleagues/friends).

Disadvantages

Most deep learning algorithms aren't great at analyzing graph data because it is not structure like a sentence/image/numerical data is.

How We Make It Work

CNN Implementation

First they analyzed how CNN works. Notice how it convolves around the extracted information in an image. When it convolves, it is taking in a relationship between a pixel and its relatives (8 if it is non-border pixel).

Introduction to Message Passing

The thing earlier about the relationships between the pixels in the CNN. Guess what, it's a grid like graph. Ever node is a pixel connected to 8 neighbor pixels.

  Each person is assigned a color.
  For the first round they pass on their message (their color) all of the immediate nodes around them.
  Imagine that your color mixes with the colors that you receive from your neighbor.
  This is essentially how the algorithm determines the structure of the graph.
  Every time you do a round of message passing, it's like another convolutional layer added to the algorithm for analysis.

Node Embeddings

Introduction

Embedding is numerical representation of the node's data. The embedding space the space of all possible numerical representations of the nodes.

Learning and Loss Functions

The embeddings have to be assigned based on a certain property. For example, the people who you spend the most time with chatting online should be closer to you in the social network. This creates a measurable objective which can allow the model to quantify the difference between the nodes.

Use Cases

Link Prediction

Using the node data to make a prediction about new possible edges. Example: finding new friends to add to your social network based on mutual friends and interests/hobbies

Node Classification

Use a node representation to predict something about the nodes. Example: Is the node in the social network an eldest child?

Clustering

Breaking up the embedding space into different groups based on certain properties of the nodes. Example: extrovert/introvert.

Graph Classification

Comparing two graphs (e.g. two people and their relationships).
