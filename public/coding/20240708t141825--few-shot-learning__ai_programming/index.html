Few Shot Learning

What is Few Shot Learning

Classification based on few number of samples in the support set (e.g. 4)

Few Shot learning is a kind of meta learning. Meta learning is learning to learn

Why

Purpose of the model isn't to learn what object is being classified, it is only to classify things. You don't need to know what a pangolin or admadillo is to know the difference when shown 2 images of each.

With this training, the model can tell you that two images are alike with good certainty, even without knowing what is shown in the image.
